# Gaurav Prakash Portfolio

A modern GitHub Pages portfolio for SRE, AIOps, Observability and Automation leadership.

## How to publish on GitHub Pages

1. Create a new GitHub repository named `gaurav-portfolio` or `<your-github-username>.github.io`.
2. Upload all files from this folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save and open the published GitHub Pages URL.

## Customize

- Replace `assets/profile.jpeg` with your latest LinkedIn photo if needed.
- Replace LinkedIn URL placeholders in `index.html`.
- Add your actual resume as `assets/Gaurav-Prakash-Resume.pdf`.
