# Gaurav Prakash Portfolio

React + Vite + Tailwind CSS + Framer Motion + Mermaid.js portfolio for GitHub Pages.

## Local run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to GitHub Pages
This project is configured for the repository `gaurav8502/Personal` using `base: '/Personal/'` in `vite.config.js`.

Option A: GitHub Actions or Pages build from branch after pushing source.

Option B: Manual deploy:
```bash
npm install
npm run build
npm run deploy
```

Your site URL:
https://gaurav8502.github.io/Personal/

## Replace files
- `public/profile.jpeg` — your LinkedIn photo
- `public/resume.pdf` — your resume
