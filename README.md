# Gaurav Prakash Portfolio

Static GitHub Pages portfolio for `gaurav8502/Personal`.

## How to update profile content

Edit only this file:

```text
data/profile.js
```

This site intentionally does not auto-fetch LinkedIn data. LinkedIn does not allow normal static websites to scrape or directly read live profile data without approved API/OAuth access.

## Publish

Upload all files to the root of the `Personal` repository and open:

```text
https://gaurav8502.github.io/Personal/
```
